This folder contains placeholder PNG icons referenced by manifest.json:

- icon16.png
- icon32.png
- icon48.png
- icon128.png

These placeholders are enough to load the unpacked extension.

If you want branded production icons later:

1. Replace each PNG with your own artwork at the same file name.
2. Keep square dimensions that match the file name.
3. Preserve PNG format and transparent background if possible.
